# Obesity Prediction - Tech Challenge Fase 4

Projeto de Machine Learning desenvolvido para prever níveis de obesidade,
auxiliando equipes médicas na tomada de decisão.

## Estrutura
- data: base de dados

- app: aplicação Streamlit
- dashboard: painel analítico
